# Privacy Policy

Exposé IP Address is a local macOS menu bar utility.

## Data Access

The app reads local network interface information provided by macOS so it can show:

- the primary local IPv4 address
- active Wi-Fi/LAN IPv4 addresses
- interface names such as `en0`

## Data Transmission

The app does not transmit IP addresses, analytics, telemetry, diagnostics, or user data to any server.

## Clipboard

When you choose a copy action, the selected IP address is written to your local macOS clipboard.

## Launch at Login

If you enable Launch at Login, macOS records that preference for this app through Apple’s ServiceManagement API.
